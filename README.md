# 📱 Auth Flow UI — Mobile App Signup, Login & Welcome Screens

A clean, modern **mobile app authentication flow** UI design featuring Welcome, Sign Up, Login, and Login Success screens — built with a bold purple color scheme and friendly illustrations.

🔗 **Live Design (Behance):** [https://www.behance.net/gallery/252064223/UIUX-Mobile-App-Signup-Flow](https://www.behance.net/gallery/252064223/UIUX-Mobile-App-Signup-Flow)

---

## 📌 Repository Name
`mobile-app-signup-flow-ui`

## 📝 Description
A 4-screen mobile authentication UI/UX flow (Welcome → Sign Up → Login → Success) designed for iOS, featuring social login options, email/password authentication, and a success confirmation screen. Ideal as a starter template or design reference for onboarding flows in mobile apps.

---

## ✨ Features

### 🎨 Design
- Modern, minimal **purple-accented** UI theme
- Custom **onboarding illustration** on the Welcome screen
- Consistent typography and spacing across all screens
- iOS-style status bar and native look & feel

### 🚪 Welcome Screen
- App introduction with headline and description
- Quick-access **Login** link for returning users
- Prominent **Sign Up** call-to-action button
- Social login shortcuts (Facebook, Twitter)

### 📝 Sign Up Screen
- Social network signup (Facebook, Twitter)
- Email and password input fields
- Password visibility toggle (show/hide)
- **Privacy Policy** agreement checkbox
- Redirect link to Login for existing users

### 🔐 Login Screen
- Social network login (Facebook, Twitter)
- Email and password fields with masked input
- Password visibility toggle
- **Forgot Password?** link
- Redirect link to Sign Up for new users

### ✅ Login Success Screen
- Animated-style success checkmark icon
- Personalized welcome message with user's name & email
- User profile summary card
- **Continue to Dashboard** CTA button

---

## 🖼️ Screens Included
| Screen | Description |

| Sign Up | Email/social registration form |
| Login | Email/social login form |
| Login Successful | Confirmation screen after successful login |
